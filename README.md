# test-loftschool

## Installation

1:
```sh
git clone https://github.com/Sky161/test-loftschool/
```

2:
```sh
cd test-loftschool
```

3:
```sh
npm i
```

## Run Server

```sh
gulp watch
```


For the correct degree of work required bower and gulp installed globally . To do this :
```sh
npm i -g bower
npm i -g gulp
```
